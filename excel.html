Excel Project

Key Outcomes

  • Developed strong proficiency in using Microsoft Excel for data analysis, reporting, and visualisation tasks.
  • Learned to efficiently organize and clean datasets using sorting, filtering, conditional formatting, and data validation techniques.
  • Applied powerful Excel functions including VLOOKUP, XLOOKUP, INDEX-MATCH, IF, SUMIF(S), COUNTIF(S), and TEXT functions to manipulate and analyse data.
  • Created insightful, interactive pivot tables and pivot charts to summarise large datasets and uncover key trends.
  • Utilised Excel Tables and Named Ranges to structure data efficiently and improve formula accuracy.
  • Built dynamic dashboards using form controls, slicers, and linked charts to support executive level decision making.
  • Enhanced productivity by automating repetitive tasks using macros and basic VBA scripting.
  • Visualised data trends and comparisons using advanced chart types such as combo charts, waterfall charts, and sparklines.

Skills & Tools Utilised

  • Excel Functions: Proficient in using advanced functions such as VLOOKUP, XLOOKUP, INDEX-MATCH, IF/IFS, SUMIF(S), and COUNTIF(S) for data analysis.
  • Pivot Tables & Charts: Created powerful summaries and visualisations to analyse complex datasets effectively.
  • Data Cleaning: Applied sorting, filtering, conditional formatting, and data validation to maintain data accuracy and consistency.
  • Dashboards: Built interactive dashboards using slicers, form controls, and linked visualisations for real time insights.
  • Named Ranges & Tables: Structured data efficiently to improve formula readability and reduce errors.
  • Macros & VBA: Automated repetitive processes and enhanced functionality through basic VBA scripting.
  • Advanced Charting: Visualised data using combo charts, waterfall charts, sparklines, and custom formatting for clear communication.
  • What-If Analysis: Used tools such as Goal Seek and Scenario Manager to support business decision making.
