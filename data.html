Data Cleaning

Key Outcomes

  • Developed a structured approach to data cleaning and preprocessing, ensuring high data quality before analysis.
  • Identified and handled missing, duplicate, and inconsistent data using tools like Python (Pandas), Power Query, and SQL.
  • Standardised categorical data by correcting inconsistencies and applying consistent naming conventions.
  • Applied techniques for outlier detection and treatment using statistical methods and visualisation tools.
  • Transformed raw data into clean, structured formats suitable for reliable analysis and visualisation.
  • Utilised regular expressions and string manipulation techniques to clean and standardise textual data.
  • Performed data type conversions and ensured accurate handling of date, time, and numeric fields.
  • Documented data cleaning steps to maintain data lineage and ensure reproducibility of the analysis process.

Skills & Tools Utilised

  • Python (Pandas): Cleaned and prepared large datasets using functions like dropna(), fillna(), drop_duplicates(), and data type conversions.
  • Power Query Editor (Power BI): Applied data transformation steps including filtering, grouping, merging, and replacing errors.
  • SQL: Cleaned data directly at the database level using UPDATE, CASE WHEN, and NULL handling techniques.
  • Regular Expressions (Regex): Used for cleaning and validating text data, removing unwanted characters, and standardising formats.
  • Outlier Detection: Identified and addressed outliers using statistical methods like the IQR rule and Z-scores.
  • Data Type Management: Ensured correct data types for analysis, particularly for date/time formats, numeric conversions, and categorical variables.
  • Data Documentation: Maintained clear documentation of cleaning steps to support reproducibility and transparency.

Reflection

This experience taught me that data cleaning is one of the most critical and time intensive steps in the data analysis process. I learned to apply structured methodologies to assess data quality, handle missing and inconsistent values, and prepare datasets for meaningful analysis. Through hands-on experience with tools like Pandas, Power Query, and SQL, I developed practical skills for transforming messy, incomplete data into clean, reliable datasets. This process reinforced the importance of careful documentation and maintaining data lineage, ensuring that my analysis is transparent and reproducible. I now recognize that high quality data is the foundation for accurate insights and confident, data driven decision making.
