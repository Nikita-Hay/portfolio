PowerBI

Key Outcomes

  • Developed professional, interactive Power BI dashboards to visualise key business metrics and performance indicators.
  • Connected to and transformed data from multiple sources including Excel files, CSVs, and SQL databases.
  • Utilised Power Query Editor to clean, reshape, and prepare raw data for accurate reporting.
  • Created calculated columns and measures using DAX (Data Analysis Expressions) to build advanced analytical models.
  • Designed intuitive and user-friendly dashboards incorporating KPIs, slicers, filters, and drill-through pages for interactive exploration.
  • Applied advanced visualisation techniques including trend analysis, year-over-year comparisons, and forecasting.
  • Implemented effective data storytelling by transforming complex data into clear, actionable insights for stakeholders.
  • Enhanced report interactivity with bookmarks, dynamic visuals, and custom tooltips to improve user experience.

Skills & Tools Utilised

  • Power BI Desktop: Designed and developed interactive dashboards and reports using industry-standard tools.
  • Power Query Editor: Cleaned and transformed raw datasets through data shaping, merging, and advanced transformations.
  • DAX (Data Analysis Expressions): Created calculated columns, measures, and KPIs to perform complex data analysis.
  • Data Modeling: Established relationships between tables and applied normalisation principles to build efficient data models.
  • Advanced Visualisations: Utilized custom visuals, slicers, drill-downs, and interactive charts to communicate insights effectively.
  • Data Integration: Connected and combined data from multiple sources including Excel, CSV, and SQL databases.
  • Performance Optimisation: Improved report responsiveness using data reduction techniques and optimised DAX queries.
  • Storytelling & Presentation: Applied data storytelling principles to build clear, visually appealing, and insight driven reports for decision makers.

Reflection

Working with Power BI has greatly enhanced my ability to analyse and present data in a meaningful way. I learned how to transform raw, complex datasets into interactive dashboards that provide actionable insights for business stakeholders. Through hands-on experience with Power Query Editor and DAX, I developed the skills to clean data efficiently and perform advanced calculations to uncover hidden trends. This project taught me the importance of visual design and storytelling in data analytics, ensuring that the insights I deliver are both clear and impactful. I am now confident in my ability to create professional Power BI reports that support strategic decision making and communicate complex findings in a user friendly, visually engaging format.
